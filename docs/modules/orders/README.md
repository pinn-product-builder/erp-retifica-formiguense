# Módulo de Ordens de Serviço

## 📋 Visão Geral

Gestão completa do ciclo de vida das ordens de serviço, desde a criação até a entrega ao cliente.

## 🎯 Objetivo

Centralizar todas as informações e etapas de uma ordem de serviço, integrando coleta, diagnóstico, orçamento, produção e entrega.

## 📊 Funcionalidades Principais

### Criação de OS
- Vinculação com cliente
- Dados do veículo e motor
- Componentes solicitados
- Status de garantia

### Workflow de Stages
- Recepção
- Diagnóstico
- Orçamento
- Produção
- Metrologia
- Finalização
- Entrega

### Documentação
- Histórico completo de ações
- Anexos (fotos, documentos)
- Timeline de eventos
- Comunicação com cliente

## 🔗 Integração com Outros Módulos

- **Clientes**: Dados do cliente proprietário
- **Coleta**: Solicitação de coleta vinculada
- **Operações**: Workflow Kanban
- **Orçamentos**: Orçamento aprovado
- **Metrologia**: Inspeção dimensional
